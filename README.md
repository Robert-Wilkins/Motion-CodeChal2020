### To Run The Code
Make sure you have Python3 installed and do the following:
```
$ git clone https://github.com/harrywang/self-contained-project.git
$ cd self-contained-project
$ python3 -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
$ jupyter notebook
```
